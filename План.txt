Разработать сайт для генерации грамот на основе встроенных шаблонов для одного 
или нескольких человек, если подгрузить файл с данными ФИО. После этого 
пользователь скачивает сформированные pdf-файлы. Может быть добавить возможность 
для добавления своих шаблонов.

Фронтенд: HTML, CSS, JavaScript.
Бэкенд: Python, Flask, SQLAlchemy, PostgreSQL.
Генерация PDF: WeasyPrint или pdfkit.
Обработка файлов: Pandas для анализа и обработки данных, включая чтение CSV и Excel-файлов,
Werkzeug модуль для обработки загружаемых файлов в Flask.
